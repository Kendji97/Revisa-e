Configuração inicial — Firebase
1
Crie um projeto no Firebase
Acesse console.firebase.google.com → "Adicionar projeto" → nome: revisae → desative o Google Analytics.
2
Ative o Firestore Database
Menu lateral → Firestore Database → Criar banco de dados → Modo produção → região southamerica-east1 (São Paulo).
3
Configure as regras de acesso
Firestore → aba Regras → substitua tudo por isso e clique em Publicar:rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
4
Copie as credenciais do app
No projeto → ⚙️ Configurações → "Seus aplicativos" → clique em </> → registre um app Web → copie os valores do objeto firebaseConfig.
